🧠 Mental Model

  • Spec hook – The format defines a place to execute code (e.g., PDF /OpenAction, SVG onload).
  • Renderer honour – Does the actual viewer/reader execute that hook? (Chrome PDF viewer ignores Acrobat JS, but PDF.js may be vulnerable).
  • Sink context – How is the file served? <img> strips scripts, <object> and <iframe> keep them.
